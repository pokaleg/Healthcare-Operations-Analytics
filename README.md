# 🏥 Healthcare Operations Analytics Dashboard

An end-to-end healthcare analytics project that transforms patient and hospital data into actionable business insights using Microsoft Power BI. The dashboard provides interactive visualizations of patient demographics, hospital performance, treatment outcomes, operational efficiency, and financial metrics to support data-driven healthcare decision-making.

---

## 🚀 Project Overview

Healthcare organizations generate large volumes of operational and patient data every day. This project analyzes healthcare records to identify trends in patient admissions, treatment outcomes, hospital performance, and healthcare costs through an interactive Power BI dashboard.

The dashboard enables stakeholders to monitor key healthcare metrics, evaluate hospital performance, and explore patient trends across multiple dimensions.

---

## 🛠️ Tech Stack

- **Microsoft Power BI** – Dashboard Development & Data Visualization
- **Microsoft Excel** – Data Source
- **Power Query** – Data Cleaning & Transformation
- **DAX** – KPI Calculations & Measures

## 🔄 Project Workflow

### 1. Data Collection
- Imported healthcare operational dataset from Microsoft Excel.

### 2. Data Preparation
- Cleaned and transformed data using Power Query.
- Validated data types and prepared fields for analysis.

### 3. Data Modeling
- Built relationships between healthcare data tables.
- Created calculated measures using DAX.

### 4. Dashboard Development
- Designed an interactive Power BI dashboard.
- Implemented KPI cards, charts, slicers, and drill-down analysis.

### 5. Business Analysis
- Evaluated hospital performance.
- Analyzed patient demographics.
- Explored treatment and billing trends.
- Identified operational patterns supporting strategic decision-making.
